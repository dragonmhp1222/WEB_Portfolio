## DESIGN SYSTEM

### なぜApple・大手が高級に見えるか（4原則）

```
1. 余白が多い   → 要素間に空気を入れる。詰め込まない
2. 色を絞る     → 背景・テキスト・アクセントの3色のみ。装飾に色を使わない
3. フォントが強い → デザインより文字の大きさ・ウェイトで魅せる
4. 動きが少ない → アニメは入りより消えを丁寧に。飾りのアニメ禁止
```

### スペーシング（8px基準・全要素がこの倍数に乗る）

```
コンポーネント内余白:  16px  (p-4)
カード内余白:          24px  (p-6)
セクション内余白:      40px  (py-10)
セクション間余白 SP:   64px  (py-16)
セクション間余白 PC:   96px  (py-24)
ヒーロー余白:         128px  (py-32)
```

### タイポグラフィ

```
Display: 64〜72px / 700 / tracking -0.04em  （大型ヒーロー）
H1:      48px     / 700 / tracking -0.03em
H2:      36px     / 600 / tracking -0.02em
H3:      24px     / 600 / tracking -0.01em
Body:    16px     / 400 / line-height 1.8
Small:   14px     / 400 / line-height 1.6

日本語: Noto Sans JP
英数字: Inter
```

### カラートークン（業種別）

#### テック・SaaS・Apple風
```
bg:       #FFFFFF / #F5F5F7（Appleのグレー）
text:     #1D1D1F（Appleのブラック）
accent:   #0071E3（Appleブルー）
section:  白とダーク(#000000)を交互に
feature:  グラデーションテキスト可（bg-clip-text）
```

#### 大手コーポレート・BtoB
```
bg:       #F8F8F6（オフホワイト）
text:     #111111
accent:   #1E3A5F（ネイビー）
border:   #E2E2E2（薄いグレーで区切る）
tone:     左揃え・シリアス・写真より図表
```

#### 美容・ライフスタイル
```
bg:       #FAF7F2（クリーム）
text:     #2C2C2C
accent:   #C9A96E（ゴールド）or #D4A5A5（ローズ）
tone:     写真大きく・テキスト少なく・余白で魅せる
font:     英語部分はセリフ体（Playfair Display）
```

#### 飲食・カフェ
```
bg:       #FFFFFF or #0D0D0D（ダーク系）
text:     #111111 or #F5F5F5
accent:   #C0392B（深紅）or #D4A017（ゴールド）
tone:     料理写真全面・予約CTAを最優先
```

## 売れる構造（セクション順・変えない）

```
① FV         → キャッチ（誰に・何を）+ CTA + ビジュアル
② 数字の実績  → 「制作〇件・満足度〇%・納期〇日」（架空でも入れる）
③ 悩み提示   → ターゲットの課題に共感
④ 解決策     → サービス説明
⑤ 制作物     → スクリーンショット + ライブリンク
⑥ 制作の流れ → 3〜5ステップ（不安除去）
⑦ 料金目安   → 必ず記載（不明=離脱）
⑧ FAQ        → 「問い合わせしない理由」を全部潰す（最低5問）
⑨ 最終CTA   → ベネフィット付き
```

### CTAコピー原則
```
❌「お問い合わせはこちら」
✅「無料で相談する（返信24時間以内）」
✅「まずは構成案を見てみる」
```

```yaml
画像:           next/image（priority指定をFVのみに）

```

### アニメーションパターン（これ以外使わない）

```tsx
// フェードアップ（基本形）
const fadeUp = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0, transition: { duration: 0.5, ease: [0.25, 0.1, 0.25, 1] } }
}

// 複数要素を順番に出す
const stagger = {
  animate: { transition: { staggerChildren: 0.1 } }
}

// 使い方（ビューポートに入ったら発火）
<motion.div
  initial="initial"
  whileInView="animate"
  viewport={{ once: true, margin: "-60px" }}
  variants={fadeUp}
/>
```

## 受注できる提案文の型（案件ごとにここだけ変える）

```
はじめまして。LP・Webサイトの新規制作を行っております。

本件を拝見し、[案件内容の1行要約] に対して
具体的な提案ができると思いご連絡しました。

▼ご提案内容
・[この案件に特有の課題] を解決する構成設計
・スマホ最適化（モバイルファースト）
・SEOを意識したコーディング
・Vercelへのデプロイまで一貫対応

▼実績
・bemikata.com（SaaS営業支援アプリ、個人開発・運用中）
・ポートフォリオ：[URL]

現在、実績拡大のため特別価格にて受付中です。
その代わり、完了後にポートフォリオへの掲載許可とフィードバックをお願いしています。

まずはお気軽にご相談ください。
```

**ここだけ毎回変える（2行で十分）：**
```
・[案件内容の1行要約]
・[この案件に特有の課題]
```

これをやるだけで「テンプレ感」がなくなり、通過率が上がる。
